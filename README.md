# Neuro-trello

To run this project you need to have `docker` and `docker-compose` installed and then run this commands:

-   Build containers

```bash
make build_dev
```

-   Run containers

```bash
make run_dev
```

-   Stop containers

```bash
make stop
```

-   Down containers

```bash
make down
```

---

The frontend server is on port `3000` and the backend server is on port `3500`.
